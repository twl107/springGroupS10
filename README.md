## ✨ Key Features

### 1. 주문 및 결제 프로세스 구현
* **PortOne API** 연동을 통한 실시간 결제 시스템 구축
* 주문서 작성부터 결제 완료, 주문 이력 저장까지의 원스톱 결제 프로세스 구현

### 2. 커머스 핵심 비즈니스 로직
* **세션 기반 보안 인증** 시스템을 활용한 회원 관리
* 장바구니 관리, 상품 조회, 주문/배송 관리 등 커머스 필수 기능 설계 및 개발

### 3. 콘텐츠 및 데이터 관리
* 파일 업로드/다운로드 기능을 포함한 자료실 및 공지사항 게시판 구축
* 원활한 정보 공유를 위한 게시판 환경 구현

## 📈 Learning & Achievement
* **Full-Cycle 개발 역량:** DB 설계부터 프론트엔드까지 전 과정을 주도하며 데이터의 전체적인 흐름을 명확히 파악했습니다.
* **사용자 중심의 설계:** 단순한 기능 구현에 그치지 않고 사용자가 더 편리하게 시스템을 이용할 수 있는 환경을 고민하며 서비스 완성도를 높였습니다.
* **실무 기술 적응력:** 외부 API 연동 및 레거시 환경에서의 데이터 처리 방식을 익히며 실무적인 문제 해결 능력을 배양했습니다.
